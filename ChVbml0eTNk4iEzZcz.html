由3d电影开始

电影院中的3d电影常常被大家各种吐槽，但不得不承认，带上眼镜之后字幕好像确实变成了立体的了:P。当然在不带眼镜的时候看上去就有点模糊了。具体原理参考wiki

如何在Unity3d中实现类似效果

  • 首先必须要有两个摄像机
  • 然后将每个摄像机上的图像渲染到texture
  • 然后合并两张texture

这里介绍一下Unity中的一个重要功能Render Texture

使用这个功能我们就能将摄像机所捕捉的画面绘制到Texture上,而我们只需要将两张texture按一定的方式合并即可。

下面附上简单的shader代码： ~~~~{c} Shader "Custom/combine" { Properties { _MainTex ("Base (RGB)", 2D) = "white" {} _Tex2 ("Base (RGB)", 2D) = "white" {} } SubShader { Tags { "RenderType"="Opaque" } LOD 200

    CGPROGRAM
    #pragma surface surf Lambert

    sampler2D _MainTex;
    sampler2D _Tex2;

    struct Input {
        float2 uv_MainTex;
    };

    void surf (Input IN, inout SurfaceOutput o) {
        half4 c = tex2D (_MainTex, IN.uv_MainTex);
        half4 c2 = tex2D (_Tex2, IN.uv_MainTex);

        //这里直取了了两张texture的颜色平均值，仅作实验使用
        o.Albedo = (c.rgb + c2.rgb)/2;    
        o.Alpha = c.a;
    }
    ENDCG
} 
FallBack "Diffuse"

}

~~~~
