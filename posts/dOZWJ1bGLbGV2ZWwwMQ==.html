Level01关卡介绍：

这一关主要是要利用有漏洞的程序(源代码在下面列出)，来达到执行任意程序的目的。少年，能否找到就看你的本事了。 使用level01的账号登陆系统（密码也是level01） 相关文件可以在/home/flag01中找到。

关卡原址

#include <stdlib.h>
#include <unistd.h>
#include <string.h>
#include <sys/types.h>
#include <stdio.h>

int main(int argc, char **argv, char **envp)
{
  gid_t gid;
  uid_t uid;
  gid = getegid();
  uid = geteuid();

  setresgid(gid, gid, gid);
  setresuid(uid, uid, uid);

  system("/usr/bin/env echo and now what?");
}

先让我们去看看所谓的相关文件吧

细心的你是否已经发现flag01的权限上有SetUID位，而其拥有者又是flag01。这说明了我们在执行flag01的时候拥有flag01用户的权限。这是一条重要的线索。

让我们来了解下UID(User identifier)的概念吧

虽然我们平常使用的用户都是字符串形式的，但是Unix系的系统在内部都是通过UID（一个整数）来区分用户的。字符串和实际的UID的映射存储在/etc/passwd中。

在一个进程中，通常包含了三种类型的UID：

  • EUID(Effective user ID)
  • SUID(Saved user ID)
  • RUID(Real user ID)
