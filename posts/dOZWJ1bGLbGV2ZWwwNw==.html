Level07关卡介绍：

在这一关中flag07写了一个perl程序来ping远程主机，看是否能从web server Ping通。

使用level07的账号登陆系统（密码也是level07）。

关卡原址

看看是否在/home/flag07下能发现一些线索：

thttpd.conf分析：

  • 端口为7007
  • 默认目录是/home/flag07
  • 并不使用chroot
  • 启动后将会以flag07身份运行
  • **.cgi的文件将被认为是cgi程序（可以访问）

index.cgi分析

#!/usr/bin/perl
use CGI qw{param};
print "Content-type: text/html\n\n";
sub ping {
     $host = $_[0];
     print("<html><head><title>Ping results</title></head><body><pre>");
     @output = `ping -c 3 $host 2>&1`;
     foreach $line (@output)
     {
          print "$line";
     }
     print("</pre></body></html>");
}
# check if Host set. if not, display normal page, etc

ping(param("Host")); #从URL中解析出Host，并传入ping函数

如何将Host传入进去呢：

赶紧掏出万能下载神器wget： 

wget http://127.0.0.1:7007/index.cgi?Host=www.baidu.com
得到
