想用迅雷下载的缘由

自从Unity更新到了4.7之后，下载Unity的方式也从原本的直接链接下载改为了使用安装器下载。但是安装器的下载速度显然是比不上使用了p2p技术的迅雷的。亲自尝试，安装器的下载速度不到50k。。因此也就萌生了用迅雷下载的想法。

如何获取下载地址

下面方法适用于mac系统，window方法应该也是类似的。 首先下载Unity官方的安装包，只有2.5MB。 然后双击安装包出现安装界面。 选择显示包内容

依次打开Contents/Resources/settings.ini

[settings]
ini_url1=http://netstorage.unity3d.com/unity/5b98b70ebeb9/unity-5.0.0f4-osx.ini
ini_url2=http://download.unity3d.com/download_unity/5b98b70ebeb9/unity-5.0.0f4-osx.ini
ini_url3=http://beta.unity3d.com/download/5b98b70ebeb9/unity-5.0.0f4-osx.ini
ini_url4=http://files.unity3d.com/bootstrapper/5b98b70ebeb9/unity-5.0.0f4-osx.ini

然后打开浏览器，尝试打开文件中的url,这里我选取了第一个url

注意：可能有的url会无法打开，但我们只需要其中一个可以打开就行了

下面是url显示的内容

[Unity]
title=Unity 5
description=Unity Editor
url=MacEditorInstaller/Unity.pkg        //注意这里是Unity的下载地址
md5=586029ed099dac2b84c7b382001ec39c
install=true
mandatory=false
size=1768773803
installedsize=5142701000
[WebPlayer]
title=Web Player
description=Development Unity Web Player
url=MacWebPlayerInstaller/WebPlayer.pkg     //这里是Web Player的下载地址
md5=cee57b1f7faaae61e80c1b34a54d308a
install=true
mandatory=false
size=2775118
installedsize=6911000
[StandardAssets]
title=Standard Assets
description=Unity Standard Assets for easily getting started building projects in Unity
url=MacStandardAssetsInstaller/StandardAssets.pkg  //这里是StandardAsset的下载地址
md5=894cf7bf1ac26d5ada1affb5f4026087
install=true
mandatory=false
size=208122389
installedsize=203181000
[Example]
title=Example Project
description=Unity 5 Example Project
url=MacExampleProjectInstaller/Examples.pkg  //这里是示例工程的下载地址
md5=51aeeeb1beeb393fd1e7568b180b3c3b
install=false
mandatory=false
size=347281113
installedsize=616155000

最后将url组合一下就可以找到真正的下载地址了。 组合方法：

复制打开上述文件的url，将最后的unity-xxx.ini替换为网页中的url的值即可

例如WebPlayer的下载地址为： http://netstorage.unity3d.com/unity/5b98b70ebeb9/MacWebPlayerInstaller/WebPlayer.pkg

Happy Downloading~
