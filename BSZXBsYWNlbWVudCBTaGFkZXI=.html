偶然之间发现了Unity中存在Replacement shader这个东西。让我们来看看它到底有什么用吧。

用处：

  • 用于在运行时根据Tag替换相应的shader
  • 可用于debug和某些特效

食用方法：

主要通过调用 Camera.RenderWithShader或Camera.SetReplacementShader 来实现替换

函数原型：

public void SetReplacementShader(Shader shader, string replacementTag);

替换过程：

  • 若replacementTag为空，则所有物体都会使用替换的shader来渲染
  • 若replacementTag不为空
    • 实际渲染使用的shader是根据tag查找的
    • 若当前shader不存在该tag，则物体不会被渲染
    • 若用来替换的shader中不包含该tag则物体也不会被渲染
    • 只有两者都有相应tag的shader的时候才能渲染

测试过程：

原shader（单纯渲染白色）：

Shader "Custom/RenderWhite" {
    SubShader {
        Tags { "RenderType"="Opaque" "MyTag" = "RenderWhite”}//注意这里指定了MyTag

        Pass{
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag

            float4 vert(float4 v:POSITION) : SV_POSITION{
                 return mul(UNITY_MATRIX_MVP, v);
            }

                fixed4 frag() : COLOR{
                 return fixed4(1,1,1,1);
             }
            ENDCG
        }
    } 
    FallBack "Diffuse"
}

替换用的shader（单纯渲染绿色）：

Shader "Custom/ReplacementShader" {
    SubShader {
        Tags { "RenderType"="Opaque" "MyTag" = "RenderWhite"}//注意这里指了定相同的MyTag

        Pass{
            CGPROGRAM
            #pragma vertex vert
            #pragma fragment frag

            float4 vert(float4 v:POSITION) : SV_POSITION{
                 return mul(UNITY_MATRIX_MVP, v);
            }

                fixed4 frag() : COLOR{
                 return fixed4(0,1,0,1);
             }
            ENDCG
        }
    } 
    FallBack "Diffuse"
}

替换脚本：

using UnityEngine;
using System.Collections;

public class testReplacementShader : MonoBehaviour {
    public Shader rs;

    void Start () {
        Camera.main.SetReplacementShader(rs, "MyTag”);//这里指定了需要替换掉的Tag

    }

    void Update () {
        if(Input.GetKey(KeyCode.Space))
        {
            Camera.main.ResetReplacementShader();
        }

    }
}
